# Open-Source Roadmap

## Near-term

- deepen Perturb-seq across multiple public studies
- add one public CELLxGENE or Tabula Sapiens state-bearing subset
- add ENCODE signal summarization for marker loci
- add Reactome pathway subset ingestion
- improve benchmark baselines

## Medium-term

- compact pseudo-cell builders for bulk sources
- richer spatial neighborhood summarization from HuBMAP
- open TCGA molecular subset path where legally supportable
- constraint profiles backed by Reactome, OmniPath, and Recon-like priors

## Longer-term

- controlled-data adapters for UK Biobank and deeper TCGA use
- large-scale COD release packaging
- stronger action models and world/policy/value training loops
