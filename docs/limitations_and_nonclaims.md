# Limitations And Non-Claims

## What COD currently does not claim

- It does not claim exact cross-source single-cell joins where sources do not support them.
- It does not claim that metadata-only sources are equivalent to direct molecular state.
- It does not claim that weak action labels are definitive biological truth.
- It does not claim public metadata access is equivalent to full raw-data support.
- It does not claim that controlled-access resources are fully ingested.

## Current deepest public path

The strongest open path in this repository is still the public-subset program:

- perturbation metadata plus selected public marker-gene expression from GEO Perturb-seq
- public metadata/context from HCA, ENCODE, HuBMAP, TCGA
- public signaling priors from OmniPath

That is useful, but it is not yet the end state of a biologically deep operational dataset.
